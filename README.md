# 🎾 Tennis Score Simulation App

## 📌 Description

Tennis Score Simulation App adalah aplikasi berbasis Python yang digunakan untuk mensimulasikan sistem skor dalam pertandingan tenis.

Aplikasi ini dibangun menggunakan konsep Object-Oriented Programming (OOP) untuk memodelkan jalannya pertandingan, mulai dari game, set, hingga match secara keseluruhan.

---

## 🚀 Features

* Simulasi pertandingan tenis secara real-time
* Pemilihan pemain dari daftar pemain WTA
* Menampilkan profil lengkap pemain
* Fitur "Last 20 Points" untuk melihat riwayat poin terakhir
* Statistik pertandingan seperti pertandingan profesional

---

## 🛠️ Tech Stack

* Python
* Django
* HTML
* CSS

---

## ▶️ How to Run

1. Clone repository:

   ```bash
   git clone https://github.com/achmad-irfan/tennis-score-simulation-app.git
   ```

2. Masuk ke folder project:

   ```bash
   cd tennis-score-simulation-app
   ```

3. Jalankan server:

   ```bash
   python manage.py runserver
   ```

4. Buka browser dan akses:

   ```bash
   http://127.0.0.1:8000/
   ```

5. Pilih 2 pemain dan mulai simulasi pertandingan
